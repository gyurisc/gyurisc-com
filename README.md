# gyurisc-com
My personal site with links and projects and occasional blogs. Using NextJs 13 and some great template from the vercel template page.

## Templates to consider 

- Tom Hirst - https://www.tomhirst.com/links/
- [Portfolio starter kit](https://vercel.com/new/gyurisc-s-team/templates/next.js/portfolio-starter-kit) from Nextjs 
- [NextJs Portfolio](https://vercel.com/new/gyurisc-s-team/templates/next.js/nextjs-portfolio) with  Blog 
- [OneLink](https://vercel.com/new/gyurisc-s-team/templates/nuxt/onelink) allows you to create landing pages.
